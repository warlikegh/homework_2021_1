inverse(array, stat) → {Array}

The "inverse" function reverses the order of the elements in an array. If a number is passed to the function as the second argument, then all the elements of the array are rearranged except for the first few (the number depends on the number). If the number is negative, then the elements at the end of the array remain in place
Parameters:
Name Type Default Description
array Array An array of elements (won`t change)
stat number 0 A number of elements to skip. If the number is negative, then the elements at the end of the array remain in place.
